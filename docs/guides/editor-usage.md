# 编辑器使用教程

本文档介绍 Quantum Lowcode 编辑器的使用方法。

## 界面概览

编辑器主界面包含以下区域：

```
┌─────────────────────────────────────────────────────────────┐
│  Header (顶部工具栏)                                         │
├────────────┬────────────────────────────┬──────────────────┤
│            │                            │                  │
│  Sidebar   │       Sandbox             │  Props Editor   │
│  (左侧面板) │       (画布区域)           │  (右侧属性面板)   │
│            │                            │                  │
│  - 组件列表 │                            │  - 组件属性      │
│  - 图层    │                            │  - 样式配置      │
│            │                            │  - 事件配置      │
├────────────┴────────────────────────────┴──────────────────┤
│  Footer (状态栏)                                            │
└─────────────────────────────────────────────────────────────┘
```

## 基础操作

### 添加组件

1. 在左侧组件面板中找到目标组件
2. 拖拽组件到画布区域
3. 组件将被添加到画布中

### 选择组件

- **单击**：选中单个组件
- **Ctrl/Cmd + 单击**：多选组件
- **Ctrl/Cmd + A**：全选

### 移动组件

- 拖拽选中的组件进行移动
- 使用键盘方向键微调位置

### 调整大小

- 拖拽组件边框调整大小
- 拖拽角落进行等比缩放

### 删除组件

- 选中组件后按 `Delete` 键
- 或右键选择"删除"

### 复制/粘贴

- `Ctrl/Cmd + C`：复制
- `Ctrl/Cmd + V`：粘贴
- `Ctrl/Cmd + D`：复制并粘贴

## 撤销/重做

- `Ctrl/Cmd + Z`：撤销
- `Ctrl/Cmd + Shift + Z`：重做

## 右键菜单

右键点击组件可打开操作菜单：

- 复制
- 粘贴
- 删除
- 上移一层 / 下移一层
- 置于顶层 / 置于底层
- 对齐

## 画布操作

### 缩放

- 工具栏缩放滑块
- `Ctrl/Cmd + 滚轮`：缩放

### 预览

点击"预览"按钮查看最终效果。

### 清空

点击"清空"按钮清空画布。

## 组件属性配置

在右侧属性面板中配置选中组件的属性：

### 1. 基本属性

配置组件的基本属性，如文本、颜色等。

### 2. 样式配置

配置组件样式：
- 尺寸
- 间距
- 边框
- 背景
- 定位

### 3. 事件配置

配置组件事件处理函数。

## 数据源配置

### 添加数据源

1. 切换到"数据源"面板
2. 点击"添加数据源"
3. 选择数据源类型（基础/HTTP）
4. 配置数据源参数

### 使用数据源

在组件属性中使用数据源：

```
${dataSource.fieldName}
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Delete | 删除选中 |
| Ctrl+C | 复制 |
| Ctrl+V | 粘贴 |
| Ctrl+Z | 撤销 |
| Ctrl+Shift+Z | 重做 |
| Ctrl+A | 全选 |
| 方向键 | 微调位置 |
| Ctrl+D | 复制粘贴 |

## 保存与导出

### 保存到本地

点击"保存"按钮保存到 localStorage。

### 导出配置

点击"导出"按钮下载 JSON 配置文件。

### 预览

点击"预览"按钮查看最终效果。
