# app 实例

## api

`setConfig(config)`: 设置schemas  
`setPage(field)`: 设置页面  
`getPage(field)`: 查询页面  
`deletePage`: 删除页面  
`registerComponent`: 注册组件   
`unregisterComponent`: 删除组件  
`resolveComponent`: 获取组件  
`request`: 请求方法
`emit`: 调用方法,例如 `emit('datasourceId:functionName', params)`, `emit('nodeID:functionName', params)`
